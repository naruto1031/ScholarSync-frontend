# Scholar Sync: 課題管理システム

## 概要
「課題管理システム」は、効率的なタスク管理とチームのコラボレーションを支援するウェブアプリケーションです。このシステムは、教員の方が簡単に課題を作成でき、生徒は簡単に、課題提出や現状の課題の提出状況管理をすることができます。

## URL
https://www.scholar-sync.systems/
